# Portfolio Projects

This repository contains the source code for my personal portfolio websites, showcasing different design philosophies and technical skills.

## Project Structure

### 1. Chaos Portfolio (`/chaos-portfolio`)
A highly interactive, React-based portfolio featuring:
-   **Chaos Mode**: A physics-driven, interactive playground using Matter.js.
-   **Professional Mode**: A clean, scrollable overlay for professional information.
-   **Tech Stack**: React, Vite, Framer Motion, Matter.js, Tailwind CSS.

### 2. HTML Portfolio (`/html portfolio`)
A classic, static HTML/CSS portfolio.

## Getting Started

To run the **Chaos Portfolio** locally:

1.  Navigate to the directory:
    ```bash
    cd chaos-portfolio
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
